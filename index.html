jQuery Mambo

Mambo is a jQuery plugin that let you easily create rounded badges using canvas.

You could use it for creating colorfoul badges with labels, display percentages or both. It was created because I wanted to avoid the mess of using images made ad-hoc for every percentage and to play for the first time with canvas!

Plugin usage


$("#full-features").mambo({
  value: 65, // percentage value
  text: {
    label: "Cr", // Label used, use "" to avoid value displaying
    displayValue: true, // if we want to display the value and the label
    textColor: "#ccc" // color for label and label
  },
  internalCircle: {
    fill: '#5965E5', // color used to fill the internal circle
    line: '#282D66' // color used for the internal circle border
  },
  percentage: {
    drawUnfilled: true, // display the remaining part of the percentage border
    unfilledColor: "#999EDD", // color for the remaining part of percentage border
    color: "#393F90" // color used for our percentage
  },
  drawShadow: true // draws a little shadow effect on the edge of the badge
});

Plugin defaults

OptionTypeDefaultDescription
valuefloatnoneThis is a mandatory option
text.labelstringnoneThis is a mandatory option, you can use an empty string to avoid label display
text.displayValuebooleantrueLet you choose if you want to display or not the percentage value in the main badge circle
